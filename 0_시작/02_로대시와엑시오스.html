프론트엔드 개발 용어

ES(ECMAScript)

  • 국제 표준화 단체인 ECMA International에서 제정한 자바스크립트 표준 사양
  • 현재 사용되는 대부분의 브라우저는 ES5(ECMAScript 2015)를 지원함

폴리필 (Polyfill)

  • 오래된 웹 브라우저에서도 ECMAScript의 최신 메소드와 객체를 사용할 수 있게 해주는 라이브러리

문자열 표기 방법

명칭 표기 방법 비고
케밥 케이스 my-comp 단어를 하이픈으로 구분
스네이크 케이스 my_comp 소문자로 시작하여 단어들의 연결 부위에 있는 첫 글자만 대문자로 입력
카멜 케이스 myComp 소문자로 시작하여 단어들의 연결 부위에 있는 첫 글자만 대문자로 입력
파스칼 케이스 MyComp 각 단어의 첫 글자는 대문자로 입력
