DOM 변경을 자동화하는 데이터 바인딩을 하려면, 템플릿에서 사용하는 모든 데이터를 리액티브 데이터로 정의해야 함

리액티브 데이터

  • Vue.js에서 가져오거나(get), 설정하거나(set), 훅(hook) 처리가 등록되어 반응하는 데이터

리액티브 데이터 정의

컴포넌트의 data 옵션에 문자열이나 객체 등의 데이터를 정의하면, 인스턴스 생성시 모두 리액티브 데이터로 변환된다.

    
      var app = new Vue({
        el: '#app',
        data: {
          msg: 'Vue.js' //msg는 변화를 감지할 수 있게 됨.
        }
      });
    
  
    
      var state = {cont: 0}
      var app = new Vue ({
        el: '#app',
        data: {
          state: state
        }
      });
      state.count++  // state.count 는 리액티브 데이터
